## Getting Started

First, clone the repo and run the project:

```bash
git clone https://github.com/limchinfeng/cat304-group38.git

npm ci 
npx prisma generate
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
